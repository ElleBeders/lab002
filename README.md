# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего римские обозначения цифр от I до IX, получить их десятичные представления.

### Алгоритм решения

Программа запрашивает у пользователя римские цифры от 1 до 9 и записыват их в список из строк. С помощью функции List.map устанавливает соответствия между римскими и арабскими цифрами, создавая новый список. Программа берёт строку из первого списка, сопоставляет с арабскими числами и вносит в новый список элемент, эквивалентный элементу из старого списка. Если соответствия нет, программа внесёт в новый список значение 0. Оба списка выводятся пользователю на экран.

### Тестирование

Скриншоты результата работы программы

## Задача 2

### Текст задачи

Список содержит десятичные цифры. Составить число из чётных цифр.

### Алгоритм решения

Программа запрашивает у пользователя арабские цифры. С помощью функции List.fold программа берёт по одному значению из списка, проверяет его на соотесвие условию (х%2=0), если соответствует, то программа умножает накопленное чило на 10, чтобы повысить его разрядность и добавляет цифру из списка. Если цифра не подошла под условие, программа продолжает перебор, пок список не кончится. После выполнения функции List.fold, программа вывдит на экран исходный список и полученное число.

### Тестирование

Скриншоты результата работы программы
